## Description

[Nest](https://github.com/nestjs/nest) framework CLI generated the boilerplate of this project
Project Cavanas: https://docs.google.com/document/d/15Lc7hHb8ZGcZ_Uoa24ebXXdxYbupp6e8RhkgygpGw4I/edit#

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
